# codename

> 此仓库的灵感来源于[SmartHypercube/codename](https://github.com/SmartHypercube/codename)，是其实现的Node.js版本。

Generates random codenames for your projects.

在项目立项时，命名往往是最纠结的，这个库可以帮你生成一个随机且符合规范的名称。

## Example
- NK-23
- LM-89
- OG-29
- NK-23
- LM-8

## Installation

```bash
npm install -g @vkboo/codename
```
## Usage

```bash
codename
```
